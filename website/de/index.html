Direct answers

Short answers for fast evaluation.

This section gives decision-makers, integrators and answer engines the canonical definitions of DEALIoT, DEALHost and DEALData.

What is DEALIoT?

DEALIoT is a production-oriented IoT data platform for MQTT ingestion, Kafka transport, schema governance, stream processing, backfill and operational evidence.

What is DEALHost?

DEALHost operates the DEAL suite on hardened infrastructure: Kubernetes, monitoring, backups, runbooks, security and support boundaries.

What is DEALData?

DEALData structures catalogues, data products, controlled sharing, FAIR exports, Zenodo, OpenAIRE and Data Act or Data Governance Act evidence.

How do I start an evaluation?

The recommended path is to run the local smoke test, choose one representative field stream, scope a 30-day pilot and score the evidence before production.

Which use cases fit?

The suite targets research teams, precision agriculture, connected livestock, industrial IoT, public projects and integrators that must industrialize field data.

Is the platform compliance-certified?

The repository provides compliance evidence, workflows, topics and templates. These assets support audits, but do not replace organization-specific certification.

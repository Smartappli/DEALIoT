# Adopters And References

This file tracks public adoption references for DEALIoT, DealHost, and DealData.

No public adopters are listed yet. This is intentional: the repository must not invent traction or publish private pilots without written approval.

## How To Add A Public Reference

Open an adopter story issue with:

- Organization name approved for publication.
- Contact or maintainer who approved the reference.
- Use case summary.
- Deployment target.
- DEAL components used.
- Measurable outcome or operational lesson.
- Any confidentiality limits.

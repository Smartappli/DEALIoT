# OpenAIRE Discovery Approval Template

Use this template before exposing DEALIoT metadata to OpenAIRE, EOSC or another external discovery
catalogue.

| Check | Status |
|---|---|
| Dataset exists in `governance.dataset.catalog` |  |
| DMP exists in `governance.data_management_plans` |  |
| Dataset landing page or DOI is stable |  |
| Metadata access right matches dataset classification |  |
| Restricted, internal, personal or mixed datasets are not marked open access |  |
| GDPR legal basis, DPIA decision and retention are reviewed |  |
| Data Act and DGA checks are reviewed where applicable |  |
| Licence, rights statement and reuse notes are approved |  |
| OpenAIRE PROVIDE or OAI-PMH source owner is identified |  |
| Export evidence is recorded in `governance.repository.exports` |  |

Record the approved artefact in `compliance.legal.dossier` with
`artifact_type=openaire_approval`.

# Data Sharing Agreement Checklist

Use this checklist for DGA-mediated access, Data Act third-party sharing and research transfers.

| Check | Status |
|---|---|
| Recipient identity and role are verified |  |
| Purpose, scope and duration are limited |  |
| Dataset ids and DMP ids are listed |  |
| Personal-data role and GDPR basis are recorded |  |
| Processor or joint-controller terms are attached |  |
| Trade-secret and confidentiality safeguards are attached |  |
| Security controls and access revocation are defined |  |
| No own reuse by the intermediation layer is documented where DGA applies |  |
| Onward transfer and publication are restricted |  |
| Audit logging and evidence topic are specified |  |
| Breach and misuse notification contacts are defined |  |
| Signature / approval evidence URI is recorded |  |

Record the approved artefact in `compliance.legal.dossier` with
`artifact_type=data_sharing_agreement`.

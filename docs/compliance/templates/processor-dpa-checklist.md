# Processor DPA Checklist

Use this checklist before involving hosting providers, research partners, application providers or
other processors.

| Check | Status |
|---|---|
| Controller, joint-controller or processor role is recorded |  |
| Processor instructions are documented |  |
| Sub-processors are listed and approved |  |
| Confidentiality obligations are included |  |
| Security measures are attached |  |
| Assistance with rights requests and DPIA is covered |  |
| Breach notification deadline is defined |  |
| Return/deletion at end of service is defined |  |
| Audit and evidence rights are defined |  |
| International transfer mechanism is recorded |  |

Record the approved artefact in `compliance.legal.dossier` with
`artifact_type=processor_dpa`.
